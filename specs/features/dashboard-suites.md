# Dashboard Suites


## User Story

As a tester,
I want to see all the test suite runs on my dashboard,
So that I know what has run.


## Design

The Dashing home page should show all test suite runs it has in reverse chronological order from top to bottom.
It should show them in a list or table.
It should show the suite's name and start time.
If the suite is still running, it should be marked as "In Progress".
If the suite has finished, it should be marked as "Complete", and its end time should be displayed.
